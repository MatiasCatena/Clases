  1. Generar un archivo llamado funciones.php:
  1. Definir una función mayor() que reciba 3 números y devuelva el mayor.

  4. Modificar tabla para que si recibe un sólo parámetro utilice el número 100.

  1. Generar un archivo llamado incluir.php que retorne un echo con el siguiente string “Soy incluir.php”.

  2. Reemplazar include por require. ¿Qué sucede?
  3. Generar en la misma carpeta un archivo saludo.php que solamente haga echo “Hola mundo”. ¿Qué sucede ahora con el punto a y el punto b?

  5. En el archivo incluir.php reemplazar todos los include por include_once. ¿Qué sucede?

Probar llamar a varias funciones y ver cuánto vale $funcionesEjecutadas en cada momento. ¿Esto está permitido? ¿Qué sucede?
